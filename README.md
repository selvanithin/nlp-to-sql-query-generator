# NLP to SQL Query Generator

This project demonstrates a simple approach to converting natural language into SQL queries using Python.

## Features
- Basic natural language understanding
- SQL query generation
- Simple rule-based logic

## Example

Input:
"Show users older than 25"

Output:
SELECT * FROM users WHERE age > 25;

## Future Improvements
- Integrate with PostgreSQL
- Use machine learning for better accuracy
- Add schema awareness
